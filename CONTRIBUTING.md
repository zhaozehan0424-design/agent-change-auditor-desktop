# Contributing

## Checks

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\check-desktop.ps1
```

## Pull Requests

- Keep UI changes simple and testable.
- Update `preview.png` if the layout changes.
- Update `CHANGELOG.md` for user-visible changes.
- Do not commit private project paths, generated audit logs, or secrets.
